# Chào Mừng Đến Blog Của Bạn!

Đây là trang docs đầu tiên. Bạn có thể thêm nội dung dễ dàng:

- **Nhấn mạnh**: Dùng **bold** hoặc *italic*.
- Danh sách: 
  - Item 1: Làm quen GitHub.
  - Item 2: Tạo repo mới.
- Link: [Repo mẫu](https://github.com/JaxVN/jaxvn-blog).
- Hình ảnh: ![Docusaurus](../static/img/docusaurus.png) (ghi chú: Upload img vào `static/img/` nếu cần hình khác).

> **Lưu ý**: Save file là preview tự refresh sau 2 giây!

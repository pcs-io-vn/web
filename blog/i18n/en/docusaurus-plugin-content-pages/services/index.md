# Services

PCS Vietnam offers three main service categories:

---

## 🖥️ [Remote IT Support](/en/services/remote-it-support)
Fix software issues, install and configure your computer remotely — no need to leave home. If on-site inspection is needed, we arrange pickup and return via Grab (Ho Chi Minh City only).

## 🔧 [Hardware Repair](/en/services/hardware-repair)
Diagnose and repair physical issues: hard drives, RAM, battery, display. Pickup and return via Grab (Ho Chi Minh City only).

IT support for small and medium offices. Coming soon.
